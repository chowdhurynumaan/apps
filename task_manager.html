User Guide

1 Getting Started

Welcome to Kanban Task Manager! Organize your tasks across three columns: To Do, In Progress, and Done.

2 Creating Tasks

  • Click the + button in the To Do column
  • Enter task name (required)
  • Set priority: High, Medium, or Low
  • Choose a due date
  • Add optional tags and notes
  • Click Save Task

3 Moving Tasks

  • Tap any task card to open status options
  • Choose: To Do, In Progress, or Done
  • Task will instantly move to the selected column

4 Deleting Tasks

Click the trash icon on any task card to permanently delete it.

5 Board Modes

  • Personal Board: Only you can see your tasks
  • Shared Board: Tasks are visible to all users
  • Switch modes from the menu (☰ icon)

Mobile Tip

Swipe left and right to view all three columns on your phone!
